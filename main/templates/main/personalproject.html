Module Guide

Nova has loads of built in functions to control its functions. To use them write:

from nova import module

where module is one of the following.

  • Drive
  • Sense
  • Hear
  • Sound
  • Screen
  • CV

drive = Drive()


drive.forward(speed, time)


drive.backward(speed, time)


drive.left(angle)


drive.right(angle)
